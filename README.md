
# INFINI SIGNAL SITE

## 🚀 Så deployar du via Vercel

1. Gå till [vercel.com](https://vercel.com)
2. Logga in med GitHub
3. Klicka “New Project”
4. Välj detta repo efter du laddat upp till GitHub
5. Vercel bygger allt automatiskt – klart!

## 🛠️ Starta lokalt

```bash
npm install
npm run dev
```
